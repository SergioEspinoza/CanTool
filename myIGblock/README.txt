Este modelo de bloque IG es sobre el que estaba construyendo el modelo de edicion de grupos.
Aunque el nombre es "GroupEdit", realmente en este punto estaba siendo una imitacion
del bloque IG de CANalyzer "Custom Messages" (la pestania de "CAN", no la "Standard").

Este bloque tenia demasiadas columnas que no eran relevantes para la modalidad de edicion de
grupos / scripts en el dialogo de edicion de scripts (editscriptdialog.cpp).
